Chenmu Zhang

Postdoctoral Researcher, Rice University

Building autonomous agents for computational materials science

I develop LLM-based agents that autonomously write code, run simulations on supercomputers, and analyze results -- replacing months of manual workflow with natural-language instructions.

My agent, MatClaw, uses a code-first architecture with retrieval-augmented generation to achieve ~99% API accuracy across domain libraries, and a four-layer memory system for coherent multi-day workflows.

Paper GitHub

Grounded in first-principles physics

This work builds on years of research computing how electrons scatter in realistic materials -- the domain expertise that gives the AI work its depth.

Electron-phonon
Quadrupole scattering and high-mobility 2D semiconductors.
Electron-boundary
Surface scattering in nanoscale metals and graphene-Cu.
Electron-environment
Dielectric screening in van der Waals heterostructures.
Electron-defect
Defect-limited mobility in transition metal dichalcogenides.
